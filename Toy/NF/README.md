# 介绍

1. 大部分代码来源于[NBA项目](https://github.com/ANLAB-KAIST/NBA/tree/master/elements)，参考了dpdk的源代码。
2. include 里面是相关的头文件定义。
3. element 里面是各个 NF，每个NF按照论文分成几个element，每个element对应一个头文件。
